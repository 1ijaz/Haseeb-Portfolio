Werwolf Game

It was a role-based game, developed for iOS using swift language. In this game we have roles like "Werewolf, Witch, Hunter, Villager, Seer", every role has different powers. e.g. during the night time Werewolves will vote to kill a villager and during the daytime, villagers will vote to kill a werewolf. The witch can poison one person as well as save one person. Hunter will shoot one person on death, and Seer can see one person's role every night.
